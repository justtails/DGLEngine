Программа для работы с паками DPC формата движка. 
Программа позволяет создавать паки DPC, записывать в них любые файлы. 
Паки являются статическими, по этому редактировать их содержимое после создания 
- невозможно. Паки использую ZIP сжатие и криптуют содержимое. Для удобства
можно работать с компайлером паков используя BAT скрипты. Например,  что бы 
запечатать все содержимое папки "Data" в "Data.dpc" в BAT файле нужно прописать:
"DPC Compiler.exe" data > Data.dpc
